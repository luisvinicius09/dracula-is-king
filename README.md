# Dracula Is King

Code snippets designed to showcase the consistency and beauty of the [Dracula](https://draculatheme.com/) color theme across multiple programming languages.

## Purpose

Each file contains a single, screen-sized snippet (~55 lines) that exercises as many syntax features as possible — classes, functions, types, loops, literals, and more — to demonstrate how Dracula distributes color evenly and consistently.

## How to Use

1. Open this repo in **VS Code** with the [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula) theme installed
2. Open any snippet file
3. Screenshot it

## Languages

- [x] TypeScript — `main.ts`
